Academic Projects

  • Health Monitoring Cloud of Things System

    - Built Azure-based IoT system for real-time ECG signal tracking, preventing ”Atrial fibrillation” disorders.

    - Implemented the UART protocol to facilitate data acquisition from Arduino to Raspberry Pi, followed by transmission to the local machine.

    - Handled data flow from sensors to the middleware via MQTT.

    - Engineered a server-side architecture with Jakarta EE, employing MongoDB for data storage.

    - Developed a Progressive Web App (PWA) for sensors and state monitoring, with email notifications using SMTP.

    - Implemented robust security measures, including HTTPS/HSTS, OAuth2PKCE, and JWT.

    - Established a streamlined DevOps pipeline for seamless deployment from the Dev branch to the Azure server.

    Technologies: Python(Networkx, gymnasium, stable-baseline3, GNN), Django, Javascript, Docker


  • Car-Image Analysis Project

    - Implemented image verification using InceptionV3 and Vision Transformer (VIT) models to differentiate between AI-generated and real images.

    - Fine-tuned models including ResNet and EfficientNet to assess accident severity, categorizing damage levels as minor, moderate or severe.

    - Utilized YOLOv8 to detect specific damage areas on vehicles, leveraging the Car Damage COCO Dataset.

    - Deployed the models in a Streamlit application for straightforward user interaction.

    Technologies: Python, PyTorch, Streamlit


  • Edge-Optimized Bike Station Monitoring System

    - Created a Kafka producer for efficient data publishing, employing an open-source API for data generation.

    - Implemented Apache Spark consumer for distributed data processing.

    - Integrated Elasticsearch for decentralized data storage; Kibana for visualization.

    - Deployed applications with Docker for portability, leveraging Docker Compose to manage multiple services.

    Technologies: Kafka, Spark, Python, Elasticsearch, Kibana, Git, Docker


  • Image Search Engine

    - Extracted image features using Inception V3 and LBP from the Flickrphotos dataset stored in Elasticsearch.

    - Employed ElasticKNN to retrieve similar images based on user input.

    - Packaged the application with a Docker Compose file for streamlined deployment.

    Technologies: Elastic Stack, Python, Inception V3, LBP, Streamlit, Docker
